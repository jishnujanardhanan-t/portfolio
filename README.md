# Portfolio Website

A personal portfolio website built to showcase Salesforce development projects, technical skills, and hands-on experience.  
The website features a clean UI with a sticky scrolling effect to highlight content smoothly and improve recruiter experience.

---

## Overview

This portfolio website serves as a central place to present:
- Salesforce development projects
- Technical skill set and hands-on experience
- Practical work with Apex, LWC, and Salesforce automation
- GitHub and professional profile links

---

## Key Features

- Sticky scrolling effect for smooth section transitions
- Dedicated sections for projects, skills, and profile summary
- Responsive layout for desktop and mobile devices
- Clean and minimal UI optimized for recruiter review
- Easy-to-update project and content structure

---

## Projects Highlighted

- **HR Management System (HRMS)**  
  Salesforce-based application for managing employees, leave requests, and attendance using Apex, Triggers, Test Classes, Flows, Validation Rules, Approval Processes, Reports, Dashboards, and Lightning Web Components (LWC).

- **Real Estate Management System**  
  Custom Salesforce application to manage properties, client interactions, and listings using Apex triggers, automation, validation rules, and reports.

- **Portfolio Website**  
  Personal portfolio website built to present Salesforce projects, skills, and experience, featuring a sticky scrolling effect and responsive design for improved user experience.

---
